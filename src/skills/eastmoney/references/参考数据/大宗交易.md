## 大宗交易（Block Trades）

工具：`get_block_trades`（`src.tools.block_trades_tool.BlockTradesTool`）
描述：东财通过免费免鉴权 datacenter 报表接口（`RPT_DATA_BLOCKTRADE`）发布每笔 A 股大宗交易——相对收盘价折溢价成交的场外协议成交。每条记录含成交价、量、相对当日收盘的折溢价，以及挂单的买/卖营业部（"席位"）。大宗交易流向是机构吸筹/派发的观察信号。只读。
市场：A 股（.SH / .SZ / .BJ），港股/美股不在此报表内。
限速：经共享 `eastmoney` per-host 节流层；东财按源 IP 限流。

<br>

**端点**

`https://datacenter-web.eastmoney.com/api/data/v1/get`，reportName = `RPT_DATA_BLOCKTRADE`。

<br>

**输入参数（工具入参）**

名称 | 类型 | 必选 | 描述
---- | ---- | ---- | ----
code | str | Y | 带后缀的 A 股 symbol，如 `600519.SH` / `000001.SZ` / `830799.BJ`。先经 `resolve_secid` 校验，仅接受 market 0/1（SZ/BJ/SH）。
days | int | N | 以今天为止的回溯日历天数窗口，clamp 至 [1, 365]，默认 30。

<br>

**端点查询参数**

名称 | 取值 | 描述
---- | ---- | ----
reportName | `RPT_DATA_BLOCKTRADE` | 报表名
columns | `TRADE_DATE,SECURITY_CODE,SECURITY_NAME_ABBR,CLOSE_PRICE,DEAL_PRICE,PREMIUM_RATIO,DEAL_VOLUME,DEAL_AMT,BUYER_NAME,SELLER_NAME` | 请求列
filter | `(SECURITY_CODE="<code>")(TRADE_DATE>='start')(TRADE_DATE<='end')` | 个股 + 日期窗口
sortColumns / sortTypes | `TRADE_DATE` / `-1` | 按日期降序
pageNumber / pageSize | `1` / `200` | 单页上限 200 条
source / client | `WEB` / `WEB` | 来源标识

<br>

**返回字段**（源字段 → 输出键）

源字段 | 输出键 | 类型 | 描述
---- | ---- | ---- | ----
TRADE_DATE | trade_date | str | 成交日
SECURITY_NAME_ABBR | name | str | 简称
CLOSE_PRICE | close_price | float | 当日收盘价
DEAL_PRICE | deal_price | float | 大宗成交价
PREMIUM_RATIO | premium_ratio | float | 相对收盘折溢价率
DEAL_VOLUME | deal_volume | float | 成交量
DEAL_AMT | deal_amount | float | 成交额
BUYER_NAME | buyer_seat | str | 买方营业部
SELLER_NAME | seller_seat | str | 卖方营业部

信封：`{"ok": true, "market": "china_a", "source": "eastmoney", "data": {"code","days","count","records":[...]}}`。

<br>

**调用范例**

```python
from src.tools.block_trades_tool import BlockTradesTool

print(BlockTradesTool().execute(code="600519.SH", days=30))
```
