## 融资融券（Margin Trading / RZRQ）

工具：`get_margin_trading`（`src.tools.margin_trading_tool.MarginTradingTool`）
描述：东财公开 datacenter 暴露交易所发布的个股每日融资融券（融资融券）数据：融资余额、融资买入额、融券余额、RZRQ 合计余额，每个交易日一行。可回答"这只股票杠杆有多高？"。只读、免凭证。
市场：A 股（SH / SZ），融资融券仅覆盖 A 股。
限速：经共享 `eastmoney` per-host 节流层；东财按源 IP 限流。

<br>

**端点**

`https://datacenter-web.eastmoney.com/api/data/v1/get`，reportName = `RPTA_WEB_RZRQ_GGMX`（个股每日融资融券明细），按 `SCODE`（裸代码）过滤。

<br>

**输入参数（工具入参）**

名称 | 类型 | 必选 | 描述
---- | ---- | ---- | ----
code | str | Y | A 股代码。接受裸六位（`600519`）、带后缀（`600519.SH` / `000001.SZ`）、交易所前缀（`sh600519`）。仅 A 股。
days | int | N | 返回最近 N 个交易日，默认 30，上限 250。

<br>

**端点查询参数**

名称 | 取值 | 描述
---- | ---- | ----
reportName | `RPTA_WEB_RZRQ_GGMX` | 报表名
columns | `ALL` | 返回全列
filter | `(SCODE="<code>")` | 个股过滤
sortColumns / sortTypes | `DATE` / `-1` | 按日期降序
pageNumber / pageSize | `1` / `<days>` | 分页
source | `WEB` | 来源标识

<br>

**返回字段**（源字段 → 输出键）

源字段 | 输出键 | 类型 | 描述
---- | ---- | ---- | ----
DATE | trade_date | str | 交易日（取前 10 位 `YYYY-MM-DD`）
RZYE | financing_balance | float | 融资余额
RZMRE | financing_buy | float | 融资买入额
RZCHE | financing_repay | float | 融资偿还额
RQYE | short_balance | float | 融券余额
RQYL | short_volume | float | 融券余量
RZRQYE | margin_total_balance | float | 融资融券合计余额

信封：`{"ok": true, "market": "a_share", "source": "eastmoney", "data": {"code", "rows": [...]}}`（最新在前）。

<br>

**调用范例**

```python
from src.tools.margin_trading_tool import MarginTradingTool

print(MarginTradingTool().execute(code="600519.SH", days=30))
```
